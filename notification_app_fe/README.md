Notification App

Features

* Create notifications
* Frontend API integration
* Logging middleware
* Error handling
* Console logging

Technologies Used

* Next.js
* JavaScript
* Fetch API

Folder Structure

* notification_app_fe
* logging_middleware

How to Run

npm install
npm run dev

Logging Middleware

The logging middleware sends logs to the evaluation server using:

* stack
* level
* package
* message

API Used

POST /api/notifications

Output

Notifications are created successfully and logs are generated.
